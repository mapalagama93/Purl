# Better way to do API calls
Simply Purl is a command line tool to makes API calls. Sample Purl spec as below.

**sample.yaml**

    Method: GET
    Endpoint: https://dummyjson.com/products/1
   
Run with following command
  

    purl -f sample.yaml
